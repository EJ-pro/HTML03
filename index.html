HTML tag kind

  • <title> Tag : Text to be entered in the title bar
  • <link> Tag : Used primarily to connect external css files
  • <script> Tag : Used to connect external js files or enter javascript code
  • <body> Tag : the part corresponding to the main text
  • <p> Tag : Use to break up paragraphs
  • <br> Tag : a new line
  • <b> Tag : bold text
  • <h> Tag : Corresponding to the title
  • <hr> Tag : Draw a straight line
  • <div> Tag : Making a box
